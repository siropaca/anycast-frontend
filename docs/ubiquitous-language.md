# ユビキタス言語集

プロジェクト全体で統一する用語の定義。

| 英語 | 日本語 | 説明 |
|------|--------|------|
| Channel | チャンネル | ポッドキャスト番組。エピソードをまとめる単位 |
| Episode | エピソード | 個別の配信回。音声コンテンツの最小単位 |
| Script | 台本 | エピソードの台本。AI で生成される |
| Script Line | 台本行 | 台本を構成する個別の発話行 |
| Character | キャラクター | 台本の話者。ボイスと紐づく |
| Voice | ボイス | 音声合成に使用する音声モデル |
| BGM | BGM | エピソードのバックグラウンドミュージック |
| Studio | スタジオ | クリエイター向けの管理画面 |
| Playlist | 再生リスト | ユーザーが作成するエピソードの再生リスト |
| Follow | フォロー | ユーザー間のフォロー関係 |
| Reaction | リアクション | エピソードへの高評価 |
| Playback History | 再生履歴 | エピソードの再生履歴 |
