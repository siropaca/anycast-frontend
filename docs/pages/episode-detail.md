# エピソード詳細 (`/episode/:episodeId`)

リスナー向けのエピソード再生ページ。認証不要でアクセスできる。

## 表示内容

- **ヘッダー**
  - エピソードアートワーク
  - エピソードタイトル
  - チャンネル名（チャンネル詳細ページへのリンク）
  - 配信者名（ユーザーページへのリンク）
  - 公開日・再生時間
- **再生プレイヤー**
  - 再生 / 一時停止
  - シークバー
  - 再生速度変更
  - 音量調整
- **エピソード説明文**
- **関連エピソード**
  - 同じチャンネルの他のエピソード

## アクション

| アクション | 未ログイン | ログイン済み |
|------------|:----------:|:------------:|
| 再生 | ✅ | ✅ |
| 再生リストに追加 / 削除 | ログインへ誘導 | ✅ |
| お気に入り / お気に入り解除 | ログインへ誘導 | ✅ |
| 共有 | ✅ | ✅ |

## API エンドポイント

- `GET /channels/{channelId}/episodes/{episodeId}` — エピソード詳細取得
- `POST /episodes/{episodeId}/playlist` — 再生リストに追加
- `DELETE /episodes/{episodeId}/playlist` — 再生リストから削除

## SEO

- インデックス対象（`robots: { index: true }`）
- OGP: エピソードタイトル・説明文・アートワークを設定
