# チャンネル詳細 (`/channel/:channelSlug`)

リスナー向けのチャンネル公開ページ。認証不要でアクセスできる。

## 表示内容

- **ヘッダー**
  - チャンネルアートワーク
  - チャンネル名
  - 配信者名（ユーザーページへのリンク）
  - チャンネル説明文
  - フォロワー数・エピソード数
- **エピソード一覧**
  - 公開済みエピソードを新着順で表示
  - エピソードのアートワーク・タイトル・公開日・再生時間
  - 各エピソードはエピソード再生ページへのリンク

## アクション

| アクション | 未ログイン | ログイン済み |
|------------|:----------:|:------------:|
| エピソード再生 | ✅ | ✅ |
| フォロー / フォロー解除 | ログインへ誘導 | ✅ |
| 共有 | ✅ | ✅ |

## API エンドポイント

- `GET /channels/{channelId}` — チャンネル情報取得
- `GET /channels/{channelId}/episodes` — エピソード一覧取得

## SEO

- インデックス対象（`robots: { index: true }`）
- OGP: チャンネル名・説明文・アートワークを設定
